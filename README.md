# 🎓 Study-Mate AI

**An AI-driven learning platform designed to transform dense study material into interactive mind-maps and actionable insights.**

## 🚀 Overview

Study-Mate AI is a full-stack SaaS platform that leverages LLMs (Large Language Models) to assist students in doubt resolution and visualization. It features a robust backend to manage AI costs through a credit-based system and secure payment processing.

## 🛠 Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS, ShadCN UI 


* **Backend:** Node.js, Express.js 


* **Database:** MongoDB 


* **Integration:** Razorpay (Payments), LLM API (AI Logic) 


* **Auth:** Cookie-based session management with Middleware 



## ✨ Key Features

* **AI Mind-Mapping:** Automatically generates visual mind-maps from text input using AI.


* **Dynamic Doubt Resolution:** Real-time AI integration to solve complex academic queries.


* **Credit Management System:** A custom-engineered backend logic to regulate API consumption and optimize costs.


* **Secure Payments:** Fully integrated Razorpay flow for automated monthly subscriptions and credit top-ups.


* **Responsive UI:** A mobile-first, high-performance interface built for speed and clarity.



## 🏗 Backend Architecture

* **Session Security:** Implemented secure middleware to protect user routes and account data.


* **Resource Optimization:** Designed the system to handle credit deduction logic server-side to prevent front-end manipulation.
